# 💻 [Fardeen Ahmad](https://github.com/iammfardeen) — Web Designer & Developer Portfolio

A sleek, dark-themed, and fully dynamic personal portfolio website built entirely using **HTML5, CSS3, and JavaScript**. This project is engineered to be highly responsive, lightning-fast, and visually captivating, utilizing modern design techniques like the **Space Grotesk** and **Inter** typography scales.

---

## 🚀 Features

* **Premium Dark UI:** Designed with a refined color palette utilizing CSS custom variables (`--bg`, `--card`, `--cyan`, etc.) for seamless color customization.
* **Fully Responsive Mockups:** Includes an embedded browser mockup system (`.project-card`) that previews live websites elegantly using iframes across both mobile and desktop views.
* **Dynamic Work Tabs:** Smooth switching system between **Delivered** and **Ongoing** projects powered by pure JavaScript DOM manipulation.
* **Interactive "Hire Me" Dropdown:** A clean, custom dropdown menu that instantly routes users to physical calls, WhatsApp chats, or direct emails without layout shift.
* **Clean Architecture:** Styled gracefully using advanced CSS Flexbox, Grid, CSS Transitions, and specialized backdrop filters (`backdrop-filter: blur(14px)`).

---

## 🛠️ Built With

* **Markup & Core:** HTML5, CSS3 (Custom Variables, Flexbox, CSS Grid)
* **Typography:** Google Fonts (Space Grotesk & Inter)
* **Interactivity:** Vanilla JavaScript (ES6+)
* **Optimization:** Google SEO Best Practices, SVG Data URI Favicons

---

## 📄 How to Use

Since this is a self-contained web package, setting it up locally or deploying it takes seconds:

1. **Clone the project** or download the source code.
2. Open the `index.html` file in any modern web browser to view it instantly.
3. Modify the text, images, and links to showcase your own skills, projects, and contact info.
